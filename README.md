# MiniMax TTS GUI (Windows)

このアプリは、CSVから音声ファイルを一括生成する Windows 向けの個人制作・非公式ツールです。  
利用には **各自の MiniMax API キー** が必要です。  
**配布物に API キーは含まれていません。**

## 対応OS

- Windows 10 / 11

## 1. このアプリでできること

- CSVに書かれたテキストを、音声ファイルとして一括生成できます
- 出力形式は `.mp3` です
- GUIで操作できるため、コマンド入力なしで使えます
- 必要に応じて、途中で停止できます

## 2. 使い方

1. `minimax_gui_preview.exe` を起動します  
2. `CSVを選択` で入力CSVを選びます  
3. `出力フォルダを選択` で保存先を選びます  
4. 初回実行時に API キーを入力します  
5. `実行` を押します  
6. 実行ログに `[ok] ...` が1ファイルごとに表示されます  
7. 必要なら `停止` で途中中断できます  

## 3. APIキーについて

- 利用には **MiniMax API キー** が必要です
- 初回実行時、`MINIMAX_API_KEY` 入力ダイアログが表示されます
- `このPCに保存する` を有効にすると、キーは Windows DPAPI で暗号化して保存されます
- 変更・削除したい場合は、GUIの `APIキー設定` ボタンから操作できます
- API キーは **各自で取得・設定** してください
- **他人の API キーを使ったり、共有したりしないでください**

## 4. CSVフォーマット

### 必須列

- `text`

### 推奨ヘッダー

    no,filename,text,voice_id,speed,pitch,emotion,notes,output

### 各列の意味

- `text`  
  読み上げる本文です。必須です

- `filename`  
  出力ファイル名です。拡張子は不要です

- `no`  
  `filename` が空のときに使われる番号です

- `voice_id`  
  行ごとに音声を切り替えたい場合に使います

- `speed`  
  話速を指定します

- `pitch`  
  音の高さを指定します

- `emotion`  
  感情表現を指定します

- `notes`  
  メモ用です。音声生成には使われません

- `output`  
  空でない行はスキップされます

### 出力ファイル名のルール

- `filename` があればその名前を使います
- `filename` が空なら `no` を使います
- `filename` と `no` の両方が空なら行番号を使います
- 出力ファイルは `.mp3` になります

## 5. 同梱ファイル

## 6. よくある注意点

- 初回起動時に Windows SmartScreen が表示されることがあります
- API キーが未設定だと処理は開始されません
- CSV の `text` 列が空の行は処理されません
- `output` 列に値が入っている行はスキップされます
- MiniMax 側の利用状況や制限により、処理できない場合があります
